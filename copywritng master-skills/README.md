## ⚡ How to Install

### Option 1 — Claude.ai Skills (Desktop/Web)
1. Download or clone this repository
2. Open Claude.ai → Settings → Skills
3. Add the `copywriting-skill` folder as a skill
4. The skill will now trigger automatically whenever you write copy

### Option 2 — Manual Context (Any Claude Interface)
1. Copy the contents of `SKILL.md` into your system prompt
2. Reference specific files from `references/` as needed

### Option 3 — Claude API / System Prompt
```python
with open("copywriting-skill/SKILL.md", "r") as f:
    skill_content = f.read()

system_prompt = f"""
You are an expert copywriter. Use this skill for all copywriting tasks:

{skill_content}
"""
```
